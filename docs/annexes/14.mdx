# Annex 14 — National Deployment Guide (Requires Authorization)

## 14.1 Purpose
This annex provides a **high-level, non-modifiable** National Deployment Guide for VILF deployments at country scale.

Full adaptation of this guide requires explicit written authorization from Doulab due to CC BY-NC-ND restrictions.

## 14.2 Deployment Phases (National-Level)

### Phase 1 — Diagnostic & Readiness (0–6 months)
- National ecosystem diagnostic  
- Stakeholder alignment  
- Selection of pilot institutions  
- Initial NCU formation  

### Phase 2 — Pilot Labs + First Hub (6–12 months)
- 1–3 pilot labs launched  
- One hub operational  
- IMM-P® execution cycles  
- Repository & evidence setup  

### Phase 3 — Scaling Pathway (12–24 months)
- Expansion to 3–7 labs  
- Multiple hubs functional  
- First national MEL cycle  
- Benchmarking report  

### Phase 4 — System Integration (24–36 months)
- Fully operational NCU  
- Multi-hub national model  
- Mandatory accreditation  
- Scorecard cycles institutionalized  

### Phase 5 — Long-Term Sustainability (36–48 months)
- National maturity progression  
- Integration into law or decree (optional)  
- Multi-year budget alignment  
- Continuous improvement cycles  

## 14.3 National Governance Requirements
- Formal IGF adoption  
- Accredited NCU  
- Standardized repositories across all labs  
- Annual MEL + Benchmarking report  
- Sector-specific capability strategies
