# Annex 06 — MEL Checklists & Scorecards

## 06.1 Purpose  
This annex provides standardized MEL (Monitoring, Evaluation & Learning) checklists and scorecard templates for quarterly and annual performance cycles.

## 06.2 Quarterly MEL Checklist (Labs)

### Execution Evidence  
- [ ] Pre-Discovery complete  
- [ ] Discovery evidence validated  
- [ ] Validation experiments logged  
- [ ] Prototyping artifacts stored  
- [ ] Efficiency & Scaling evidence updated  
- [ ] Continuous Improvement reflections completed  

### Documentation  
- [ ] Evidence repository updated  
- [ ] Decision records consistent  
- [ ] Risk logs updated  
- [ ] Governance escalations documented  

### Performance  
- [ ] KPI data submitted  
- [ ] Collaboration metrics recorded  
- [ ] User feedback incorporated  

## 06.3 Quarterly MEL Checklist (Hubs)

### Portfolio  
- [ ] Lab KPI submissions normalized  
- [ ] Cross-lab synthesis completed  
- [ ] Capability gaps identified  
- [ ] Domain patterns analyzed  

### Governance  
- [ ] Tier-1 escalation reviews complete  
- [ ] Documentation quality checks  
- [ ] Maturity progression analysis (IMM)  

## 06.4 Quarterly MEL Checklist (NCU)

- [ ] System-wide MEL consolidation  
- [ ] Benchmarking preparation  
- [ ] Governance health review  
- [ ] Documentation audit  
- [ ] Transparency requirements updated  

## 06.5 Annual Scorecard Template

### Domains (Weighted)

1. Strategic Alignment (15%)
2. Capability & Maturity (20%)
3. Execution Quality (30%)
4. Governance Compliance (15%)
5. Performance & Learning (20%)

### Example Scorecard Fields

| KPI | Evidence Required | Score (0–5) | Weight | Weighted Score |
|-----|------------------|-------------|--------|-----------------|
| L1 — IMM-P® Completion Rate | Stage artifacts | 4 | 0.30 | 1.20 |
| L2 — Evidence Quality Index | Repository audit | 5 | 0.20 | 1.00 |
| L3 — Governance Compliance | Decision logs | 3 | 0.15 | 0.45 |

**Final Scorecard Index (FSI):** Sum of weighted scores.
