# Annex 11 — Accreditation Rubric (Lab / Hub / NCU)

## 11.1 Purpose
This annex provides accreditation criteria for:
- Innovation Labs (Tier-0)  
- Hubs (Tier-1)  
- Network Coordination Unit (Tier-2)  

Accreditation ensures quality, governance compliance, and system integrity.

## 11.2 Accreditation Principles
- Evidence-based  
- Governance-compliant  
- Maturity-aware  
- Transparent  
- Consistent across tiers  

## 11.3 Accreditation Criteria — Innovation Labs (Tier-0)

### Category A — Governance (20%)
- IGF Tier-0 compliance  
- Documented decision records  
- Escalation logs  

### Category B — Execution Quality (40%)
- IMM-P® stage completion  
- Evidence quality  
- Experimentation rigor  
- Prototyping maturity  

### Category C — Documentation (20%)
- Repository structure  
- Versioning discipline  
- Metadata completeness  

### Category D — MEL Participation (20%)
- On-time KPI submissions  
- Quarterly reviews  
- Adoption of MEL feedback  

**Accreditation Outcome:**  
Accredited / Pending Improvements / Not Accredited

## 11.4 Accreditation Criteria — Hubs (Tier-1)

### Category A — Capability Development (25%)
- Training delivered  
- Coaching frequency  
- Capability progression scores  

### Category B — Portfolio Governance (25%)
- Normalization accuracy  
- Cross-lab synthesis quality  
- Domain expertise  

### Category C — Documentation (25%)
- Evidence quality enforcement  
- Repository hygiene checks  

### Category D — MEL Integration (25%)
- MEL review quality  
- Trend analysis  
- Benchmarking preparation  

**Outcome:** Accredited / Conditional / Not Accredited

## 11.5 Accreditation Criteria — NCU (Tier-2)

### Category A — Governance Leadership (30%)
- Decision-pathway authority  
- Arbitration effectiveness  
- Governance audits  

### Category B — Documentation Stewardship (30%)
- Architecture of repository  
- Metadata standards  
- Transparency rules  

### Category C — MEL & Benchmarking (40%)
- National MEL synthesis  
- Benchmarking reports  
- Accreditation cycle integrity  

**Outcome:** Accredited / Conditional / Not Accredited
