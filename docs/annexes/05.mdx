# Annex 05 — Governance Decision Rights Matrix (IGF)

## 05.1 Purpose  
This annex defines the **decision-rights matrix** for the three governance tiers of VILF, aligned with the Innovation Governance Framework (IGF).

## 05.2 Tier Definitions  
- **Tier-0 (Labs):** Execution and evidence generation  
- **Tier-1 (Hubs):** Portfolio governance and capability development  
- **Tier-2 (NCU):** Accreditation, system governance, transparency  

## 05.3 Decision Rights Matrix

### Table: IGF Decision Pathways (Tier-0 → Tier-1 → Tier-2)

| Decision Domain | Tier-0 (Labs) | Tier-1 (Hubs) | Tier-2 (NCU) |
|-----------------|----------------|----------------|----------------|
| Research design | Decide | Review | Oversee |
| Experimentation | Decide | Review | Oversee |
| IMM-P® evidence | Produce | Validate | Audit |
| Documentation | Produce | Enforce quality | Govern standards |
| MEL submission | Submit | Normalize | Synthesize |
| Portfolio alignment | Inform | Decide | Oversee |
| Governance escalation | Initiate | Handle | Arbitrate |
| Accreditation | Participate | Participate | Decide |
| Funding eligibility | Input | Recommend | Decide |
| Risk management | Identify | Review | Govern |

## 05.4 Usage  
This matrix must accompany:
- all governance protocols  
- MEL reviews  
- accreditation cycles  
- NCU audits
