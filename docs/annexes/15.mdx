# Annex 15 — Glossary Expansion

## 15.1 Purpose
This annex expands the glossary of VILF 1.0, providing authoritative definitions of all key concepts, aligned with MCF 2.1, IMM, IMM-P®, IGF, and MEL.

## 15.2 Terms

### **VILF — Vigía Innovation Lab Framework**
A structural framework for designing, governing, and scaling innovation lab networks.

### **VXF — Vigía Cross-Framework Architecture**
The shared structural and conceptual spine connecting VILF, VIF, VIGF, and future frameworks.

### **Tier-0 / Tier-1 / Tier-2**
Three-tier governance model:
- Tier-0: Labs  
- Tier-1: Hubs  
- Tier-2: Network Coordination Unit  

### **IMM — Innovation Maturity Model**
Capability and institutional maturity evaluation system.

### **IMM-P® — Innovation Maturity Model Program**
Six-stage structured innovation execution system:
Pre-Discovery → Discovery → Validation → Efficiency → Scaling → Continuous Improvement.

### **IGF — Innovation Governance Framework**
Decision-rights, escalation pathways, governance roles, and audit architecture.

### **MEL — Monitoring, Evaluation & Learning**
Performance, learning, and accountability system based on KPIs and scorecards.

### **Evidence Repository**
Structured documentation environment used to store, version, and audit evidence.

### **Hub (Tier-1)**
Capability engine and cross-lab governance unit.

### **NCU — Network Coordination Unit**
Apex governance body providing accreditation, oversight, synthesis, and strategic coordination.

### **Challenge Fund**
Competitive funding mechanism using evidence-based evaluation.

### **Benchmarking**
Cross-lab comparison system enabling learning, normalization, and performance differentiation.

### **Scorecard**
Weighted KPI-based performance tool used in MEL cycles.

### **Accreditation**
Annual quality assurance process validating compliance with governance, execution, and evidence standards.

### **Repository Hygiene**
Discipline required for maintaining documentation integrity, metadata completeness, and version control.

### **Capability Spine**
Shared set of training, coaching, and methodological development pillars.

### **Infrastructure Spine**
Shared documentation, platforms, and digital tools supporting VILF operations.
