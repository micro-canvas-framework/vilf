## Legend
- NONE: no citations by design (normative framework content)
- LIGHT: minimal grounding for context statements
- MODERATE: alignment references to external standards or institutions
- HEAVY: empirical or regulatory grounding (if applicable)

## Placement Rules
- Citations appear only in explanatory text, not in normative statements.
- References sections are added only to chapters that contain citations.
- Annexes may carry citations later under their own BL-16 tasks if required.
